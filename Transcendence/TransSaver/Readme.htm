Welcome!

The Transcendence Screen Saver displays furious battles between various ships of the Transcendence universe all while saving your monitor from dreaded burn-in.

Transcendence is a game of space combat and adventure in which you travel through the star systems of Human Space and encounter both friends and enemies. If you've like to play the game that this screen saver is based on please visit the Transcendence page online.

Intallation Instructions

Transcendence Screen Saver has been tested on Windows XP, Windows Vista and Windows 7. To install:

  1. Copy the files in this ZIP to a folder on your hard disk.
  2. Right-click on TransSaver.scr and select Install.
